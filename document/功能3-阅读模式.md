# 功能 3：阅读模式（Reader）

## 1. 功能概述

基于 unified 渲染管道的 Markdown 阅读视图，支持 GFM、代码高亮、图片放大、PDF/HTML 导出，通过 Cmd+E 与编辑模式无缝切换。

## 2. 涉及文件

### 渲染进程

| 文件 | 职责 |
|------|------|
| `src/components/reader/MarkdownRenderer.tsx` | 主渲染容器：异步渲染 HTML、图片 Lightbox、外链拦截 |
| `src/components/reader/ExportBar.tsx` | 底部导出按钮：PDF / HTML |
| `src/components/reader/ImageLightbox.tsx` | 图片点击放大遮罩 |
| `src/lib/markdown.ts` | unified 渲染管道 + 导出 HTML 包装 |
| `src/styles/reader.css` | 阅读模式排版样式（`.dz-reader`） |

### 主进程

| 文件 | 相关 IPC |
|------|----------|
| `electron/ipc/fileSystem.ts` | `export:pdf`（printToPDF）、`export:html`（保存 HTML） |

## 3. 实施细节

### 3.1 渲染管道

**renderMarkdown(content, workspace?) → Promise\<string\>**

```
content (markdown body, 不含 front matter)
  ↓ unified()
  ↓ .use(remarkParse)           → Markdown AST
  ↓ .use(remarkGfm)             → GFM 扩展（表格、任务列表、删除线）
  ↓ .use(remarkRehype)          → HTML AST（allowDangerousHtml: true）
  ↓ .use(rehypeFixImagePaths)   → 相对路径 → file:// 协议（仅传入 workspace 时）
  ↓ .use(rehypeHighlight)       → 代码块语法高亮（detect: true, ignoreMissing: true）
  ↓ .use(rehypeSanitize)        → XSS 清洗（允许 file:// 图片 + hljs class）
  ↓ .use(rehypeStringify)       → HTML 字符串
  ↓
  result: string (HTML)
```

**rehypeFixImagePaths 自定义插件**：
```typescript
// 遍历 HTML AST 中的 <img> 节点
// 将非 http/file/data 开头的 src 转为 file://{workspace}/{src}
visit(tree, 'element', (node) => {
  if (node.tagName === 'img' && node.properties?.src) {
    const src = node.properties.src;
    if (!src.startsWith('http') && !src.startsWith('file://') && !src.startsWith('data:')) {
      node.properties.src = `file://${workspace}/${src}`;
    }
  }
});
```

**rehype-sanitize schema**（允许 file 协议 + 代码高亮 class）：
```typescript
{
  ...defaultSchema,
  attributes: {
    ...defaultSchema.attributes,
    img: [['src', /^(https?:|file:)/, /^\//], 'alt', 'title'],
    code: [['className']],
    span: [['className']],
  }
}
```

### 3.2 MarkdownRenderer 组件

**数据流**：
```
currentContent (zustand)
  ↓ parseFrontMatter()
  { data.title, body }
  ↓
  body → renderMarkdown(body, workspace) → setHtml(html)
  ↓
  <div class="dz-reader">
    {title && <h1>{title}</h1>}
    <div dangerouslySetInnerHTML={{ __html: html }} />
  </div>
```

**图片点击**：
- `onClick` 事件委托在 `.dz-reader` 容器上
- 判断 `target.tagName === 'IMG'` → 取 `src` → `setLightboxSrc(src)`
- 弹出 `ImageLightbox` 全屏遮罩

**外链拦截**：
- 判断 `target.tagName === 'A'` → `preventDefault()`
- `href.startsWith('http')` → `window.open(href)` → 由 Electron `setWindowOpenHandler` 转到系统浏览器

**异步渲染取消**：
```typescript
useEffect(() => {
  let cancelled = false;
  renderMarkdown(body, workspace).then((result) => {
    if (!cancelled) setHtml(result);
  });
  return () => { cancelled = true; };
}, [body, workspace]);
```

### 3.3 ImageLightbox

- 固定定位全屏遮罩（`z-index: var(--z-modal)`，背景 `rgba(0,0,0,0.85)`）
- 居中显示图片（`max-width: 90vw`, `max-height: 90vh`）
- 点击遮罩关闭 / Escape 键关闭
- 图片点击 `stopPropagation`（防止穿透关闭）
- 淡入动画 150ms

### 3.4 导出功能

**ExportBar**：底部 36px 栏，右对齐两个按钮。

**PDF 导出流程**：
```
用户点击 "PDF" 按钮
  ↓ parseFrontMatter(currentContent)
  ↓ renderMarkdown(body, workspace)     → html
  ↓ wrapHtmlForExport(html, title)      → 完整 HTML 文档
  ↓ window.electronAPI.export.pdf(fullHtml)
  ↓ (主进程)
  ↓ dialog.showSaveDialog({ defaultPath: 'document.pdf' })
  ↓ 创建隐藏 BrowserWindow → loadURL(data:text/html) → printToPDF({ pageSize: 'A4' })
  ↓ fs.writeFile(filePath, pdfBuffer)
```

**HTML 导出流程**：
```
同上，但最后：
  ↓ window.electronAPI.export.html(fullHtml)
  ↓ dialog.showSaveDialog({ defaultPath: 'document.html' })
  ↓ fs.writeFile(filePath, htmlContent, 'utf-8')
```

**wrapHtmlForExport(bodyHtml, title?)**：
- 生成完整 `<!DOCTYPE html>` 文档
- 内联样式包含：排版（max-width 720px, line-height 1.8）、标题层级、代码块（#f6f8fa 背景）、表格、引用、列表
- 中文字体栈：`-apple-system, 'PingFang SC', 'Microsoft YaHei', sans-serif`

### 3.5 模式切换（Cmd+E）

**App.tsx 实现**：
```typescript
if (mod && e.key === 'e') {
  e.preventDefault();
  // 切换到阅读模式前先保存
  if (viewMode === 'edit' && isDirty) {
    await window.electronAPI.file.write(currentFile, currentContent);
    markSaved();
  }
  switchMode(viewMode === 'edit' ? 'read' : 'edit');
}
```

**条件渲染**：
```tsx
{viewMode === 'edit' ? <EditorArea /> : <MarkdownRenderer />}
```

**切换动画**：
- `animations.css` 定义 `contentFadeIn` 关键帧（opacity 0→1, translateY 4px→0, 200ms）
- 可通过 `.mode-enter` class 应用

### 3.6 阅读模式排版（reader.css）

**容器**：`.dz-reader`
- `font-family: var(--font-serif)`
- `font-size: var(--text-lg)` (15px)
- `line-height: var(--leading-loose)` (1.9)
- `max-width: var(--content-width)` (720px)
- `margin: 0 auto`
- `user-select: text`

**排版规范**：

| 元素 | 样式 |
|------|------|
| h1 | 2em, bold, margin 1.2em 0 0.5em |
| h2 | 1.5em, semibold, 底部 border-bottom |
| h3 | 1.2em, semibold |
| a | color #0066cc, 底部边框, hover 加深 |
| img | block, max-width 100%, 居中, radius-md, `cursor: zoom-in` |
| pre | bg-light, border, radius-md, padding 16px, font 0.85em |
| code (inline) | bg rgba(0,0,0,0.04), padding 2px 5px, radius 3px |
| blockquote | 左侧 3px 灰色竖线, padding 0.5em 1em, 斜体 |
| table | 100% 宽, collapse, th 有 bg-light |
| hr | 1px border-top, margin 2em |
| 任务列表 checkbox | pointer-events: none（只读） |

**暗色覆盖**：
- inline code: `bg rgba(255,255,255,0.08)`
- link: `color #6cb6ff`
- highlight.js: 单独在 `theme-dark.css` 覆盖关键字颜色
