## P0 — 必须修复

### 1. 阅读模式暗色不完整

**位置**：`src/styles/reader.css`

当前只有 1 条 `[data-theme="dark"]` 规则（inline code 背景），缺少：

| 元素 | 当前值（硬编码浅色） | 应改为 |
|------|---------------------|--------|
| 链接 | `#0066cc` | `var(--color-link)` 或 dark 覆盖 `#6cb6ff` |
| 链接 hover 边框 | `#0066cc` | 同上 |
| blockquote 竖线 | `var(--color-gray-300)` | ✅ 已用变量，无需改 |
| 表格 th 背景 | `var(--bg-light)` | ✅ 已用变量 |
| `<del>` 颜色 | `var(--color-gray-500)` | ✅ 已用变量 |

**结论**：只需在 `reader.css` 底部补充链接暗色覆盖（`theme-dark.css` 中已有部分，需确认是否生效）。

### 2. ModeSwitch 无文件时仍可点击

**位置**：`src/components/editor/ModeSwitch.tsx`

当前 `currentFile === null` 时按钮仍可点击，虽然不会报错但 UX 不好。

**修复**：无文件时隐藏 ModeSwitch 或 disable 按钮。

## P1 — 应该优化

### 3. 清理遗留 `.dragArea` CSS

**位置**：
- `src/components/editor/EditorArea.module.css` — `.dragArea` 类未使用
- `src/components/reader/MarkdownRenderer.module.css` — `.dragArea` 类未使用

ModeSwitch 已接管顶部拖拽区域，这两处是死代码。

### 4. 编辑器高度可能不撑满

**位置**：`src/components/editor/EditorExtensions.ts` 第 73、109 行

EditorView.theme 中 `'&': { height: 'auto' }` 可能导致编辑器不撑满容器。
- 父级 `.wrapper` 已有 `flex: 1`
- 但 CM6 的 `.cm-scroller` 设为 `overflow: visible`（依赖外部滚动）
- 实际效果需要在应用中验证：如果小文件时编辑区域底部大片空白无法点击聚焦，则需改为 `min-height: 100%`。

### 5. 弹窗/遮罩颜色硬编码

以下文件使用了硬编码 `rgba()` 而非 CSS 变量：

| 文件 | 行 | 值 | 建议 |
|------|----|----|------|
| `ImageLightbox.module.css` | 6 | `rgba(0,0,0,0.85)` | 可接受（全屏遮罩通用） |
| `SearchModal.module.css` | 6 | `rgba(0,0,0,0.4)` | 同上 |
| `HistoryPanel.module.css` | 6 | `rgba(0,0,0,0.4)` | 同上 |
| `SettingsPanel.module.css` | 5 | `rgba(0,0,0,0.5)` | 同上 |
| `HistoryPanel.module.css` | 131-135 | `#22863a` / `#cb2431` | diff 语义色，已有 dark 覆盖 ✅ |
| `SearchModal.module.css` | 115 | `rgba(255,200,0,0.3)` | 高亮色，已有 dark 覆盖 ✅ |
| `SettingsPanel.module.css` | 179 | `white` | toggle 滑块，应改为 `var(--bg-white)` |

**结论**：遮罩背景硬编码可接受（明暗均为半透明黑），只需修 toggle 滑块的 `white`。

## P2 — 功能增强

### 6. 侧边栏拖拽调整宽度

当前 256px 固定不可调。可添加拖拽条支持 200~400px 范围调整。

### 7. 编辑器状态栏增强

当前 FileInfoBar 只有字数 + 阅读时间 + 保存状态。需求文档还要求：
- 行号 / 列号（`L:16 C:84`）
- 编码显示（`UTF-8`）
- `MARKDOWN LIVE` 标识

### 8. 阅读模式文章头部

需求文档要求阅读模式显示：
- `CHAPTER 01 / PHILOSOPHY` 分类标签
- 阅读时间（`READING TIME: 12 MIN`）
- 日期

当前只显示 Front Matter 的 title 作为 h1。

### 9. 右侧装饰栏

需求文档编辑模式有 48px 右侧装饰栏：
- 垂直文字 `SOURCE EDITOR MODE`
- 版本号 `V.02`
- 背景 `#F3F3F4`

当前未实现。

### 10. 文件拖拽排序

侧边栏文件列表当前不支持拖拽移动文件到其他文件夹。

### 11. 应用图标

当前使用默认 Electron 图标。需求文档要求：
- 方形黑色背景 + 白色"Z"字母
- 多尺寸（16x16 到 1024x1024）
